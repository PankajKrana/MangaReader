# MangaReader

![Swift](https://img.shields.io/badge/Swift-5.8-orange?logo=swift)
![SwiftUI](https://img.shields.io/badge/SwiftUI-1.0-blue?logo=swift)
![iOS](https://img.shields.io/badge/iOS-17-brightgreen?logo=apple)

MangaReader is a modern iOS app for **browsing, searching, and reading manga online**.  
It fetches manga data from the [MangaDex API](https://mangadex.org/) and displays it with a smooth, user-friendly interface.

---

## Features

- **Home Screen**
  - Dynamic seasonal manga card
  - Latest updates with manga covers
- **Search**
  - Real-time search with live results
  - Displays manga title, cover, and status
- **Downloads**
  - Placeholder for future offline reading
- **Efficient Image Loading**
  - Powered by [Kingfisher](https://github.com/onevcat/Kingfisher)

## Tech Stack

- **Language:** Swift 5  
- **Frameworks:** SwiftUI, Combine, Kingfisher  
- **API:** [MangaDex API](https://api.mangadex.org/)  
- **Persistence:** @AppStorage for app settings
